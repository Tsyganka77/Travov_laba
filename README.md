# Сравнение удобства раскладок

> В современном мире существует множество различных раскладок. Многие считают Йцукен не самой удобной раскладкой для использования. Использование наиболее эргономичной раскладки может существенно ускорить набор текста и снизить количество ошибок. Неправильно подобранная раскладка увеличивает нагрузку на руки, что может привести к усталости или профессиональным заболеваниям, таким как синдром запястного канала.

В своей работе мы анализируем 3 раскладки:

- Йцукен
- Вызов
- Диктор
